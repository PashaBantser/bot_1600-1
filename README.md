# Telegram-бот для створення та проходження тестів

## Опис проєкту
Цей проєкт — це Telegram-бот, який дозволяє **вчителям створювати тести**, а **учням проходити їх за кодом доступу** в реальному часі. Бот підходить для навчальних занять, перевірки знань, міні-контролів та інтерактивних уроків.

Проєкт орієнтований на простоту використання та мінімальну кількість дій як для вчителя, так і для учня.

---

## Мета проєкту
- Надати простий інструмент для створення тестів у Telegram  
- Дозволити учням швидко підключатися до тесту за кодом  
- Реалізувати перевірку відповідей у реальному часі  
- Закріпити практичні навички роботи з **aiogram**, **FSM** та **callback-кнопками**

---

## Використані технології
- **Python 3.10+**
- **aiogram 3** — асинхронний фреймворк для Telegram-ботів
- **FSM (Finite State Machine)** — керування станами користувачів
- **InlineKeyboard / ReplyKeyboard** — кнопки взаємодії
- **JSON** — зберігання даних (користувачі, тести, активні сесії)
- **asyncio** — асинхронний запуск бота

## Запуск проєкту

### 1. Встановлення залежностей
```bash
pip install aiogram
```

### 2. Налаштування токена
У файлі `modules/settings.py` вкажіть токен вашого бота:
```python
TOKEN = "YOUR_BOT_TOKEN"
```

### 3. Запуск
```bash
python main.py
```

## Як користуватися ботом

### Початок роботи
- Надішліть команду `/start`
- Оберіть роль:
  - **Вчитель**
  - **Учень**

---

## Функціонал для вчителя

### Реєстрація
Якщо вчитель користується ботом уперше:
1. Введіть імʼя  
2. Номер телефону  
3. Пароль  

Дані зберігаються у файлі `users.json`.

---

### Створення тесту
1. Натисніть **«Створити тест»**
2. Вкажіть кількість питань
3. Для кожного питання:
   - Введіть текст питання
   - Введіть **перший варіант відповіді** (він завжди правильний ✅)
   - Введіть інші варіанти відповідей ❌
   - Натисніть **«Перейти до наступного питання»**
4. Після введення всіх питань тест автоматично зберігається

---

### Перегляд тестів
- Натисніть **«Переглянути тести»**
- Для кожного тесту доступні кнопки:
  - **Отримати код**
  - **Видалити тест**

---

### Запуск тесту
1. Натисніть **«Отримати код»**
2. Скопіюйте згенерований код
3. Передайте код учням
4. Натисніть **«Розпочати тест»**
5. Питання надсилаються всім підключеним учням

---

## Функціонал для учня

### Підключення до тесту
1. Оберіть роль **«Учень»**
2. Введіть код тесту
3. Введіть своє імʼя
4. Якщо код існує — учень приєднується до тесту

---

### Проходження тесту
- Учень отримує кожне питання з кнопками відповідей
- На кожне питання можна відповісти **лише один раз**
- Після відповіді бот одразу повідомляє результат:
  - ✅ правильна відповідь
  - ❌ неправильна відповідь з правильною

---

## Особливості реалізації
- Перший варіант відповіді завжди вважається правильним
- Підтримка кількох учнів одночасно
- Захист від повторної відповіді
